Quickstart

1. Ensure Docker Desktop is installed and running.
2. From the repo root:

   ./scripts/start.sh

3. Visit:
   - Juice Shop: http://localhost:3000
   - DVWA: http://localhost
   - Vulnerable API: http://localhost:4000

Notes: Use the Kali container or your host Kali instance for offensive tooling. Always isolate the lab from production networks.
