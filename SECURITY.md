Security and responsible disclosure

This project contains intentionally vulnerable code and services for learning and testing only. Follow these guidelines:

- Do not use this code or the services in production.
- Run the lab in isolated networks or local VMs.
- If you discover a security issue in the scaffold or helper scripts that could cause real-world harm (credential leakage, remote code execution against hosts outside the lab), please open an issue and mark it as a security concern. Do not publish exploit details publicly until a fix is available.
- Use environment variables to store real secrets. Avoid committing credentials to the repo.

Contact
- Open an issue in this repository, or reach out to the repository owner for sensitive disclosures.
