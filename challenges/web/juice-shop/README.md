# OWASP Juice Shop - Challenges

OWASP Juice Shop is an intentionally insecure web application that contains a range of challenges from simple to complex.

Access:
- http://localhost:3000

Tips:
- Use the 'Score Board' in the UI to see challenge progress.
- Typical attacks include XSS, broken access control, and business-logic abuse.

Mitigations:
- Use secure coding practices, input validation, context-aware encoding, and modern auth frameworks.
