# Challenges

This directory contains individual challenge descriptions and walkthroughs for the lab targets.

- web/juice-shop: OWASP Juice Shop challenge notes and hints.
- web/dvwa: DVWA challenge notes and hints.
- api/vuln-api: Custom vulnerable API exercises (XSS, command injection, insecure auth).

Each challenge folder contains a `README.md` that describes the learning objectives, attack steps, and mitigation guidance.
